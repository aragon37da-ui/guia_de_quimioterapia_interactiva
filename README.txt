# 🌐Guia de cuidados y efectos de la Quimioterapia

¡Bienvenido! Este proyecto contiene los archivos de Guia de cuidados y efectos de la Quimioterapia. Si no tienes conocimientos de programación, no te preocupes:
 esta guía está diseñada paso a paso para que puedas instalar todo lo necesario y ejecutar la página localmente en tu computadora.

---

## 📋 Requisitos Previos

Antes de comenzar, necesitamos instalar una herramienta gratuita llamada **Node.js**. Esta herramienta incluye **npm** (Node Package Manager), 
que es el sistema que se encarga de descargar de forma automática los componentes necesarios para que la página funcione.

### Cómo instalar Node.js:
1. Ve al sitio web oficial: [nodejs.org](https://nodejs.org/).
2. Descarga la versión recomendada para la mayoría de los usuarios (normalmente dice **LTS**).
3. Abre el archivo descargado y sigue las instrucciones del instalador (haz clic en "Siguiente" o "Next" a todo).
4. **Verificación:** Para asegurarte de que se instaló bien, abre la terminal o consola de tu sistema
 (en Windows busca `cmd` o "Símbolo del sistema"; en Mac busca "Terminal") y escribe el siguiente comando, luego presiona Enter:
   ```bash
   node -v
